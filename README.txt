Credit Card Payment - Sprint Boot Service
---------------------------------------------
This is a Spring Boot component that delivers a credit card payment platform backend functionality.
The purpose of this payment system is to take a user's credit card details and store into H2 database.

All services are authenticated with Basic Auth.

Requirements
- This application is on JDK zulu-21.

How to run the application:
1. Build Project: To clean target folder and build the project (clean install).
2. Copy Test Resources: To copy the relevant environment files in the target folder so that the service can
point to the chosen environment.
3. Start Application: To start the application in IntelliJ.
